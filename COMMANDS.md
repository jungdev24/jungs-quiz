## 문제 생성

### 미생성 문제 일괄 생성
```
문제생성
```
DB에서 `ai_generated=false`인 Day를 찾아 해당 과목의 설명(없으면 제목)을 기반으로 문제를 자동 생성합니다.

### 특정 과목 문제 새로 생성
```
오후반 문제 새로 만들어줘
저녁반 문제 새로 생성해줘
주말반 Day 1~5 문제 만들어줘
```
해당 과목의 기존 문제를 초기화하고 완전히 새로운 문제를 생성합니다.

### 전체 과목 문제 재생성
```
전체 문제 새로 생성
```
모든 과목의 문제를 초기화하고 새로 만듭니다.

## 참고

- 문제 생성 시 과목에 설정된 **AI 설명**과 **첨부 파일**을 참고합니다.
- 설명이 없으면 **Day 제목**(예: 현재와 현재진행형)을 기반으로 생성합니다.
- 기본값: **20일 × 10문제** (과목 설정에서 변경 가능)
- 문제 유형: 과목에 설정된 유형만 생성 (객관식, 빈칸채우기, 단어장 등)
